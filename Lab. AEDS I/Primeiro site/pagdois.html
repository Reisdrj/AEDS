Exemplos de códigos utilizando a linguagem C:

-O script abaixo lê dois números e os mostra em ordem inversa:

    
    #include <stdio.h>

        void main () {
            float num1, num2;
            puts("Insira dois valores: ");
            scanf("%f %f", &num1, &num2);
            printf("%.1f e %.1f\n", num2, num1);
        }

-Esse verifica se o número inserido é par ou ímpar:

    
        #include <stdio.h>

            void main () {
                int num;
                puts("Insira um número para saber se ele é par ou ímpar: ");
                scanf("%d", &num);
                if ((num % 2) == 0 || num == 0)
                    printf("O número %d é par!\n", num);
                else
                    printf("O número %d é ímpar!\n", num);
            }

-Nota-se que a linguagem possui seus comandos específicos para o funcionamento correto do projeto.

Explicação dos códgios:

  • #include <stdio.h>
    Inclui a biblioteca Standard Input/Output
  • void main()
    Inicializa a função principal do código(é obrigatório ter a função main)
  • int / float / char
    Inicialiação das variáves (que deve ser feita antes do uso das mesmas)
  • printf("")
    Incluso na biblioteca 'stdio.h', é responsável por "imprimir" o conteúdo dos parênteses na tela
  • scanf("")
    Incluso na biblioteca 'stdio.h', é responsável por ler e armazenar o conteúdo inserido ém uma variável.
