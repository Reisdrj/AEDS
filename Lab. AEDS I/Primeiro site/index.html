Linguagem C

A Linguagem de Programação C foi criada em 1972 por Dennis Ritchie, sendo a sucessora da linguagem "B" (que era uma forma reduzida da BCPL). Ela ainda é bastante utilizada atualmente para diversos fins, entre eles: podução de parte de sistemas operacioanis, editores de imagem, robôs.

Características da Linguagem

  • Manipulação: com a linguagem C, é mais fácil exercer a manipulação de todas as partes do sistema.
  • Portabilidade: ela pode ser rodado de forma extremamente fácil em qualquer sistema operacional.
  • Classificação: não é considerada uma linguagem de baixo nível, nem de alto nível, mas de "médio" nível.
  • Multiuso: C pode ser usado para a produção de diversos projetos.

O importante de se saber a linguagem C, é que ela é usada como base de diversas linguagens,como: PHP, C++ e C#.

Como utilizar a linguagem

Cada linguagem possui suas características específicas, e com o C não é diferente. A exemplo: para se inicializar um programa com funções comuns na linguagem, precisa-se:

  • Utilizar uma diretiva para incluir as bibliotecas com as funções básicas
  • Iniciar uma função "main" (obrigatoriamente)
  • Utilizar a identação correta
  • Utilizar os comandos específicos da linguagem

Bibliotecas(funções e exemplos)

As bibliotecas são muito utilizadas nas linguagens de programação. Elas fornecem funções adicionais, ou seja, que não são as padrões da linguagem, para complementar e facilitar o desenvolvimento de um projeto.

Exemplos de bibliotecas de C:

  • math.h
  • stdlib.h
  • string.h
  • time.h
